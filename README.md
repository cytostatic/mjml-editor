# MJML Editor

MJML Drag&Drop Editor for Visual Studio Code. This extension uses [easy-email](https://github.com/zalify/easy-email). Big thanks to its author.

## Features

* Drag&Drop editor for MJML files
* Desktop and mobile preview panel.
